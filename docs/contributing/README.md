# Contributor Docs Index

This folder contains contributor-focused documentation and templates.

Files
- testing-your-pr.md: What to run locally before pushing or opening a PR
- pull-request-template.md: Example of a high-quality PR description
- bug-report-template.md: Structured, reproducible bug report
- feature-request-template.md: Clear feature/enhancement request format with acceptance criteria

Where these guidelines come from
- Repository Makefile targets (build, lint, tests)
- .github workflows: lint, python tests, frontend unit tests, e2e
- Existing issue templates in .github/ISSUE_TEMPLATE/
- CONTRIBUTING.md and Development.md in this repo
- Exemplary PRs by @xingyaoww (#10578) and @neubig (#10606)
